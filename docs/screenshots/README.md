# Скриншоты для README

Положите сюда PNG-файлы с указанными именами — они автоматически подхватятся
в [главном README](../../README.md) (раскомментируйте блок «Скриншоты»).

| Файл | Что заснять |
|---|---|
| `user-start.png` | Экран пользователя после `/start` (заявка отправлена) |
| `admin-request.png` | Уведомление админу о новой заявке с кнопками |
| `user-key.png` | Выдача ключа пользователю (текст-ключ + `.vpn`-файл) |
| `admin-panel.png` | Панель администратора `/admin` |
| `admin-keygen.png` | Автономный ключ через `/keygen` |
| `admin-traffic.png` | Статистика трафика `/traffic` |

**Рекомендации:** ширина ~720–900px, формат PNG, тёмная тема Telegram —
скриншоты выглядят аккуратнее и единообразнее.
